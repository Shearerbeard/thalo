# Protobuf Example

A client & server demonstrating an in-memory bank account app with an aggregate and projection.

## Run Server

```bash
$ cd ./examples/protobuf
$ cargo run --bin server
```

## Run Client

```bash
$ cd ./examples/protobuf
$ cargo run --bin client
```
