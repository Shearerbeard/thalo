# Protobuf Example

A client & server demonstrating an in-memory bank account app with an aggregate and projection.

## Run Server

```bash
$ cargo run -p example-protobuf --bin server
```

## Run Client

```bash
$ cargo run -p example-protobuf --bin client
```
